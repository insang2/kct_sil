# 한국건설트레이딩 - 건설 자재 유통 및 판매 B2B/B2C 플랫폼 기획서

## 1. 프로젝트 개요 (Overview)
- **프로젝트명:** 한국건설트레이딩 (KCT) 온라인 이커머스 플랫폼
- **서비스 목적:** 건설 관련 자재의 온/오프라인 유통 및 판매 경험을 혁신하는 종합 웹 플랫폼 구축
- **주요 대상:** 일반 소비자 (B2C) 및 기업/건설현장 담당자 (B2B)
- **기술 스택 방향:** Next.js 기반으로 구축하여 빠른 페이지 속도와 검색 엔진 최적화(SEO) 확보

## 2. 사용자 타겟 분석 (Target Audience)

### 🧍 B2C (일반 소비자 및 셀프 인테리어족)
- **특징:** 소량 구매 위주. 상세한 제품 설명과 시각적 자료(시공 리뷰 등)가 구매 결정에 중요함.
- **요구사항:** 모바일 환경에서의 직관적인 UI, 간편 가입, 무통장/신용카드/간편결제 등 다양한 수단 지원.

### 🏢 B2B (건설사, 인테리어 시공업체, 현장 소장)
- **특징:** 대량 및 반복 구매 위주. 단가와 세금 증빙이 가장 중요하며, 결제 프로세스가 B2C에 비해 복잡함.
- **요구사항:** 사업자 전용 할인 단가, 대량 구매 할인(Volume Discount), 견적서 발행, 세금계산서 신청, 여러 건설 현장별 배송지 분리 관리.

## 3. 핵심 기능 (Key Features)

### 🛒 공통 쇼핑몰 기능
- **회원 시스템:** 일반/기업 회원 분리 로그인, 소셜 로그인 연동 (카카오, 네이버 등).
- **상품 관리 & 검색:** 철근, 시멘트, 공구 등 세분화된 카테고리 트리, 키워드 검색.
- **주문/결제:** 장바구니, 통합 결제 시스템 연동 (Toss Payments 또는 PortOne 고려).
- **마이페이지:** 배송 상태 조회, 주문 내역 확인, 1:1 고객 불만/오류 접수 게시판.

### 💼 B2B 특화 기능 (차별점)
1. **사업자 전용 단가표 (B2B Pricing):** 로그인된 사업자 회원 등급에 따라 일반 소비자와 다른 구매 단가가 적용됨.
2. **견적서/명세서 자동 발행:** 장바구니에 담은 상품을 바탕으로 회사 직인이 찍힌 견적서 PDF 파일 다운로드.
3. **다중 배송지 현장 관리:** 기업 회원이 구매 시, A현장, B현장 등 배송지를 별도로 지정하여 분할 배송을 요청할 수 있는 기능.
4. **간편 재주문 (Quick Re-order):** 이전 프로젝트/현장에서 구매했던 자재 리스트를 그대로 불러와 빠르게 반복 발주.

## 4. UI/UX 디자인 방향성
- **컬러 팔레트:** 산업 전반의 든든한 신뢰감을 주는 `Navy` 또는 `Deep Blue` 톤을 핵심 색상으로 사용하되, 무겁지 않은 모던한 레이아웃 구성.
- **모바일 최적화 (Responsive Web):** PC 환경의 본사 구매직원뿐 아니라, 스마트폰을 사용하는 현장 소장님들도 쉽게 발주할 수 있도록 모바일 사용성 극대화.

## 5. 기술 스택 (Tech Stack)
- **프론트엔드:** Next.js (App Router 활용), Tailwind CSS
- **상태 관리:** Zustand (장바구니 등 클라이언트 상태) + React Query (서버 상태)
- **백엔드/DB:** (초기 비용/속도 효율을 위해) Supabase 구성 권장, 또는 별도의 Node.js/Java 백엔드 서버 구축
- **인증:** NextAuth.js
- **배포 인프라:** Vercel (Front) + Supabase (DB) -> 빠른 MVP 서버리스 환경 구축

## 6. 개발 단계 제안 (Roadmap)
- **1단계 (기획/설계):** 상세 요구사항 정의, 화면 설계서(Figma Wireframe), 데이터베이스 구조 설계. 👉 *현재 단계입니다.*
- **2단계 (공통 프레임 잡기):** Next.js 프로젝트 세팅, UI 컴포넌트 시스템 구축, DB 및 인증 연동, 회원가입/마이페이지 구현.
- **3단계 (쇼핑몰/B2C MVP):** 상품 목록/상세 페이지 구현, 장바구니/주문서 로직 개발, 결제 연동.
- **4단계 (B2B 로직 고도화):** B2B 등급별 가격 변동 적용, 견적서 PDF 기능, 현장별 다중 배송 시스템 도입.
- **5단계 (관리자 페이지 & 런칭):** 대시보드, 주문/회원 관리자 기능 개발 후 최종 QA 및 서비스 오픈.
